# יצירת אייקונים פשוטה

## אם אין לך sharp מותקן, יש 3 אפשרויות:

### אפשרות 1: שימוש בשירות אונליין (הכי פשוט!)

1. לך ל-https://realfavicongenerator.net/
2. העלה את `public/logo.png`
3. הורד את כל האייקונים
4. העתק אותם ל-`public/`
5. שנה שמות לפורמט: `icon-72.png`, `icon-96.png`, וכו'

### אפשרות 2: שימוש ב-Photoshop/GIMP

1. פתח את `public/logo.png`
2. שנה גודל ל:
   - 72×72 → שמור בשם `icon-72.png`
   - 96×96 → שמור בשם `icon-96.png`
   - 128×128 → שמור בשם `icon-128.png`
   - 144×144 → שמור בשם `icon-144.png`
   - 152×152 → שמור בשם `icon-152.png`
   - 192×192 → שמור בשם `icon-192.png`
   - 384×384 → שמור בשם `icon-384.png`
   - 512×512 → שמור בשם `icon-512.png`
3. שמור הכל ב-`public/`

### אפשרות 3: התקן sharp והרץ את הסקריפט

```bash
npm install sharp --save-dev
npm run generate-icons
```

## בינתיים - אפשר להשתמש בלוגו הקיים

האפליקציה תעבוד גם בלי האייקונים המיוחדים.
היא פשוט תשתמש ב-`logo.png` הקיים.

כדי לבדוק את ה-PWA **עכשיו**, פשוט:

```bash
npm run dev
```

ופתח את האתר. תראה את כפתור "התקן אפליקציה"!
