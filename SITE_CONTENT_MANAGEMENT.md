# ניהול תוכן האתר

## סקירה כללית

המערכת מאפשרת עריכה מלאה של כל המלל והתוכן בדף הבית דרך ממשק האדמין, ללא צורך בידע טכני או עריכת קוד.

## איך זה עובד?

### 1. כניסה לממשק הניהול
- גש ל-`/admin`
- התחבר עם Netlify Identity
- לחץ על **"תוכן האתר"** בתפריט הצד

### 2. סקציות הניתנות לעריכה

#### 🎯 Hero (כותרת ראשית)
- **כותרת ראשית** - שם בית המדרש
- **כותרת משנה** - מיקום/שכונה
- **כתובת** - כתובת מלאה

#### 📖 אודות בית המדרש
- **כותרת** - כותרת הסקציה
- **פסקה ראשונה** - תיאור כללי
- **פסקה שנייה** - תיאור נוסף

#### ⭐ מה אנו מציעים
- **כותרת** - כותרת הסקציה
- **6 פריטים** - כל פריט כולל:
  - כותרת
  - תיאור
  - אייקון (מוגדר מראש)

#### 🤝 הצטרפו אלינו (Call to Action)
- **כותרת** - כותרת הקריאה לפעולה
- **תיאור** - טקסט ההזמנה

#### 🔗 גישה מהירה
- **כותרת** - כותרת הסקציה

### 3. שמירת שינויים

1. ערוך את התוכן הרצוי
2. לחץ על **"שמור שינויים"**
3. השינויים יישמרו ב-Firebase ויופיעו מיד באתר
4. אם טעית, לחץ על **"בטל שינויים"** לפני השמירה

## מבנה הנתונים ב-Firebase

הנתונים נשמרים ב-Firestore במיקום:
```
settings/siteContent
```

## קבצים רלוונטיים

### Frontend
- `src/admin/AdminSiteContent.jsx` - ממשק עריכת התוכן
- `src/pages/Home.jsx` - דף הבית שמשתמש בתוכן הדינמי

### Backend
- `netlify/functions/firebase-site-content.js` - Netlify Function לניהול התוכן

## תכונות

✅ **עריכה חיה** - כל שינוי נשמר ונטען מיד  
✅ **גיבוי אוטומטי** - localStorage כ-fallback  
✅ **אבטחה** - רק משתמשים מחוברים יכולים לערוך  
✅ **קריאה ציבורית** - כל המבקרים רואים את התוכן המעודכן  
✅ **ביטול שינויים** - אפשרות לבטל לפני שמירה  

## טיפים

1. **כתוב בבהירות** - השתמש בשפה פשוטה ומובנת
2. **שמור באופן קבוע** - אל תאבד שינויים
3. **בדוק את התוצאה** - צפה בדף הבית אחרי כל שינוי
4. **שמור עקביות** - שמור על סגנון כתיבה אחיד

## פתרון בעיות

### השינויים לא נשמרים
- בדוק שאתה מחובר
- בדוק את ה-console בדפדפן לשגיאות
- ודא שיש חיבור אינטרנט

### התוכן לא מתעדכן באתר
- נקה את ה-cache של הדפדפן
- רענן את הדף (Ctrl+F5)
- בדוק ב-Firebase Console שהנתונים נשמרו

### שגיאת אימות
- התנתק והתחבר מחדש
- בדוק שהמשתמש שלך מאושר ב-Netlify Identity
